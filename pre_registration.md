## 1. Research Questions

1. Do frontier LLMs substantially disagree on which brands to recommend for the same commercial query?
2. Is that disagreement robust to surface-level query paraphrasing?
3. Is that disagreement stable across temperature settings?
4. Does disagreement vary systematically by query category?
5. Are models internally consistent (intra-model) while externally divergent (inter-model)?

---

## 2. Hypotheses

All hypotheses below are pre-registered and confirmatory unless labeled exploratory.
Statistical threshold: alpha = 0.05 with Holm-Bonferroni correction across the 5 confirmatory tests.

### H1 (primary, confirmatory)
Fleiss kappa across 11 frontier LLMs on commercial recommendation queries is < 0.40 (fair agreement or below), indicating substantial cross-model disagreement.

### H1a (sensitivity, confirmatory)
Kappa is invariant across 3 paraphrases of each query (within 95% CI of main kappa). Paraphrase-induced variance does not exceed within-trial stochasticity.

### H1b (sensitivity, confirmatory)
Kappa is stable across temperatures T = {0.0, 0.3, 0.7} (one-way ANOVA F p > 0.05).

### H1c (primary, confirmatory)
Kappa varies systematically by query category, with information-rich categories showing higher kappa than experience-rich categories.

Pre-registered category ordering (high to low kappa):
b2b_saas > financial_services > enterprise_software > consumer_electronics > cpg > healthcare_wellness > travel_hospitality > local_services

Test: rank-order Spearman correlation between predicted and observed category kappa ranks.
Threshold: rho >= 0.50.

### H2 (exploratory)
Architecture group effect: search-augmented vs. parametric vs. non-English-corpus models cluster differently in extraction agreement.
Metrics: Cohen d, Mann-Whitney U.

### H4 (primary, confirmatory)
Intra-model Jaccard / inter-model Jaccard >= 1.5. Models are internally consistent but externally divergent.

## 12. Threats to Validity

### Internal
- Stochastic variation: mitigated by K >= 5 trials
- Temporal confounding: mitigated by >= 60-minute spacing for search-augmented models
- Query ordering effects: fixed order, identical across models
- Prompt sensitivity: addressed by paraphrase sensitivity run (H1a)

### External
- API vs. consumer interface: scope declared as API; UI validation case study (n=25) is supplementary
- Query representativeness: systematic 8-category construction; 5 queries per category
- English-only: acknowledged; do not generalize to multilingual
- Point-in-time: all runs timestamped; model updates acknowledged as limitation

### Construct
- Entity mention != endorsement: "brand mention" operationalized as name presence in response text
- Jaccard treats mentions equally: does not capture sentiment or position weight
- Self-reported attribution unreliable: meta-analytical data treated as exploratory only
